NLCI Kannada fonts
==================

This file describes the source files included with the NLCI Kannada fonts.
This information should be distributed along with the NLCI Kannada fonts and any derivative works.

master.gdl            - Graphite rules
master.fea            - OpenType rules
*.sfd files           - glyph outline data for each font
badami_unicode.xml    - mapping from legacy to Unicode as used by ttfbuilder
